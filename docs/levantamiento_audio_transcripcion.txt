Esta aplicación se basa en un proceso de retiro, entrega y almacenamiento para un servidor
portuario. Principalmente opera en base a clientes externos que envían correos solicitando
containers para ir a recibir cosas al puerto o retirar cosas del puerto. Principalmente está
en la operación porque aquí se guarda y se procesa el almacenaje de los containers ya
sea con carga o vacíos y también se despachan los containers con carga o vacíos que tenga el
depósito de almacenaje hacia el puerto para devolver o hacia el cliente para entregar dicha carga.
Ese es el proceso base principal. Todo se mueve con un documento que llaman ETA que contiene la
información del puerto, la fecha, la patente, el nombre del conductor, el depósito del contenedor,
el código del contenedor, la hora de retiro, el cliente que reserva el contenedor y la
